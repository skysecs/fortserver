# 사용 안내

## 1. 소개

이 API는 PAM 자산 계정의 비밀번호를 조회하고 RESTful 요청을 지원하며 데이터를 JSON 형식으로 반환합니다.

## 2. 환경 요구 사항

- `Java 11+`
- `HttpClient`

## 3. 사용 방법

**요청 방식**: `GET api/v1/accounts/integration-applications/account-secret/`

**요청 매개변수**

| 매개변수 이름 | 유형 | 필수 | 설명 |
|---------------|------|------|------|
| asset         | str  | 예   | 자산 이름 |
| account       | str  | 예   | 계정 이름 |

**응답 예시**:
```json
{
    "id": "72b0b0aa-ad82-4182-a631-ae4865e8ae0e",
    "secret": "123456"
}
```

## 자주 묻는 질문 (FAQ)

Q: API 키는 어떻게 얻나요?

A: PAM - 응용 프로그램 관리에서 응용 프로그램을 만들면 KEY_ID와 KEY_SECRET이 발급됩니다.
