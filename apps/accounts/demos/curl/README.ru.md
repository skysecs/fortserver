# Инструкция

## 1. Введение

Этот API получает пароли учетных записей активов PAM, поддерживает RESTful-запросы и возвращает данные в формате JSON.

## 2. Требования к окружению

- `Bash`
- `cURL`
- `OpenSSL`
- `base64`

## 3. Использование

**Метод запроса**: `GET api/v1/accounts/integration-applications/account-secret/`

**Параметры запроса**

| Имя параметра | Тип | Обязательный | Описание |
|---------------|-----|--------------|----------|
| asset         | str | Да           | Имя актива |
| account       | str | Да           | Имя учетной записи |

**Пример ответа**:
```json
{
    "id": "72b0b0aa-ad82-4182-a631-ae4865e8ae0e",
    "secret": "123456"
}
```

## Часто задаваемые вопросы (FAQ)

В: Как получить API-ключ?

О: Создайте приложение в разделе «PAM - Приложения», чтобы сгенерировать KEY_ID и KEY_SECRET.
